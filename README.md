# 风算

风算是一款黑色高级计算器风格的 Android APP。

启动页文案：

- 风算
- 一款懂你的计算管家
- by 风乱吹

## 当前版本

- 版本：1.0.11
- 包名：com.fengluanchui.fengsuan
- 最低系统：Android 6.0
- 目标系统：Android 15

## 已实现功能

- 启动页和作者署名
- 原创黑金计算器图标
- 参考图风格的稿纸计算区和五列大键盘
- 计算器页取消顶部品牌栏和上方标签
- 计算器页不再上下滑动，任意键盘布局下整屏就是计算器
- 功能入口集中在键盘底部一行和弹窗菜单
- 缩小并分级处理键盘字号，修正中文键和数字下沿裁切
- 音效支持静音、机械键盘音、中文女声三档切换
- 机械键盘音改为更清脆、更响的短敲击音
- 女声模式下按等号会播报计算结果
- 音效切换键改为喇叭、静音喇叭、女声头像图标
- 女声图标改为透明底矢量图，去除原图片方块背景
- 声音切换图标改为居中绘制，避免在按键中偏上
- 键盘数字、运算符和常用图标整体略微放大
- 修复声音切换按钮图标在部分真机上不显示的问题
- 声音切换键改为独立手绘控件，避免系统按钮图标机制导致空白
- 左上角复制键改为布局选择图标
- 新增 6 种键盘布局和布局预览选择框
- 复制、粘贴、撤销、重做、回车、删除等命令键改为图标样式
- 常用菜单中移除键盘布局入口
- 回车键图标放大并加粗
- 默认不弹出系统输入法
- 独立“文”按钮弹出文本输入框
- 可插入备注的计算草稿
- 四则运算、括号、百分号、幂、常用函数
- 光标左右移动、换行、粘贴、撤销、重做
- 复制当前数值、当前行或整个文本
- 历史保存、清空、结果求和
- 双击稿纸区切换大屏高度，双指缩放文字
- 缩放文字时稿纸横线同步缩放，文字和数字保持在同一行内
- 双击稿纸区进入全屏文本区，键盘隐藏，分享和“文”按钮退到左下角
- 全屏文本区底部显示“双击返回主界面”提示，2 秒后自动消失
- 左侧 `<` 按钮可展开分享入口
- 分享入口使用三点折线图标，弹出“请编辑待分享文本”窗口
- 分享文本可编辑，并可调用系统分享到微信、QQ 等应用
- 常规、函数、简洁三种键盘布局
- 横屏和竖屏支持
- 人民币大写金额转换
- 离线汇率估算
- 进制转换
- 小数转分数
- 常用单位换算
- 时间换算
- 关于页展示“风乱吹”

## 安装包

- `dist/风算-v1.0.11-layouts-debug.apk`
- `dist/FengSuan-v1.0.11-layouts-debug.apk`

这两个文件内容相同，英文文件名用于部分 Android 工具校验，中文文件名方便人工识别。

## 发布说明

当前 APK 是 debug 签名，适合手机安装测试。正式上架或长期分发前，应使用专门的 release keystore 重新签名。
